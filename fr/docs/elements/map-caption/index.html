<map-caption>

L'élément <map-caption> est un enfant de <mapml-viewer> et est utilisé pour définir une simple chaîne de texte qui n'est pas rendue visuellement. Le contenu devrait être lu par les lecteurs d'écran lorsque le <mapml-viewer> est focalisé, car il est utilisé pour générer la valeur de l'attribut <mapml-viewer aria-label="...">, si l'auteur n'en fournit pas. <map-caption> peut être le premier ou le dernier élément enfant de <mapml-viewer>.
