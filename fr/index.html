Documentation de <mapml-viewer>

Intégrez des visionneurs de cartes dans vos pages web, en utilisant uniquement le langage HTML.

Commençons

Code minimal

MapML donne la priorité au code minimal nécessaire pour créer des cartes Web élaborées. Aucun JavaScript n'est nécessaire, tout peut être réalisé à l'aide de HTML + éléments customisés.

Cartes accessibles

Cette suite d'éléments permet une utilisation rapide et facile des fonctions de manière accessible. L'interaction avec le clavier et la compatibilité avec les lecteurs d'écran sont constamment améliorées.

Future norme

Dans le but de faire un jour partie intégrante des navigateurs, MapML permet à toutes les cartes créées à l'aide de cette suite d'éléments personnalisés d'être facilement adaptées à la syntaxe HTML standard.

Capacités

Une analyse détailée des capacités par rapport à d'autres solutions de cartes Web est disponible dans la matrice de satisfaction des cas d'utilisation des exigences de MapML.

Participer

La suite d'éléments customisés est un projet open source. Toute personne souhaitant soumettre des modifications ou des corrections est invitée à le faire par le biais de demandes de tirage sur notre dépôt Github.

Vous pouvez également contribuer en signalant tout bogue ou problème lors de l'utilisation de la suite d'éléments par le biais d'un problème GitHub sur le même dépôt.
